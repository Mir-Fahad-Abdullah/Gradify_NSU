# Title: NSU CGPA Calculator
## Project Overview
This project introduces a CGPA calculator that is able to calculate CGPA according to the NSU (North South University) greading policy. 
**Language used:**
* Java
* JavaFX

**NSU grading policy:**

| Scores | Grade | Points |
| --- | --- | --- |
| 93+ | A Excellent | 4.0 |
| 90 - 92 | A- | 3.7 |
| 87 - 89 | B+ | 3.3 |
| 83 - 86 | B Good | 3.0 |
| 80 - 82 | B- | 2.7 |
| 77 - 79 | C+ | 2.3 |
| 73 - 76 | C Average | 2.0 |
| 70 - 72 | C- | 1.7 |
| 67 - 69 | D+ | 1.3 |
| 60 - 66 | D Poor | 1.0 |
| Below 60 | F* Failure | 0.0 |
