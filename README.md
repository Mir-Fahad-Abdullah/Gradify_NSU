# Title: NSU CGPA Calculator
## Project Overview
This project introduces a CGPA calculator according to the NSU (North South University) greading policy.

**NSU grading policy:**

| Scores | Grade |
| --- | --- |
| 93+ | A Excellent |
| 90 - 92 | A- |
| 87 - 89 | B+ |
| 83 - 86 | B Good |
| 80 - 82 | B- |
| 77 - 79 | C+ |
| 73 - 76 | C Average |
| 70 - 72 | C- |
| 67 - 69 | D+ |
| 60 - 66 | D Poor |
| Below 60 | F* Failure |
